# Sportify
Fitness Center Management and Appointment System


Brief Project Description:

This project aims to develop a Gym (Fitness Center) 
Management and Appointment System using ASP.NET Core MVC. 
The system will be capable of managing gym services, coaches' 
areas of expertise, member appointments, and AI-based exercise 
recommendations. This will allow for monitoring both coaches' 
productivity and members' personalized fitness experiences.

Technologies to be Used in the Project: 
• ASP.NET Core MVC (Current LTS version should be preferred) 
• C# • SQL Server / PostgreSQL 
• Entity Framework Core (ORM), LINQ 
• Bootstrap 5 (for theme design) 
• HTML5, CSS3, JavaScript, jQuery
